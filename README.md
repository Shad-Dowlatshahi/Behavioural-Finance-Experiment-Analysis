# Behavioural-Finance-Experiment-Analysis
This project investigates how the framing of economically equivalent investment opportunities influences decision-making, drawing on principles from prospect theory. 

Traditional economic theory predicts that rational investors should make identical decisions when faced with equivalent payoffs. Prospect theory, however, suggests that the way this information is framed can significantly influence behaviour. 

In this project, I analysed the decisions made by the 150 participants and, as predicted by prospect theory, the majority of those facing the loss frame chose the risky option (around 65%).
